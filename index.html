Portafolio

  • Sistema de Inventario de Productos Electrónicos con Python y Django

    Desarrollé E-InvSys como parte del curso de base de datos 1. Es un sistema de inventario con Python y Django para tiendas de productos electrónicos. Su objetivo es gestionar existencias, ventas y proveedores. Proporciona una plataforma eficiente y fácil de usar para un seguimiento preciso de inventario y facilitar ventas.

  • Mr-Pharma - Aplicación de Inventario de Medicamentos

    Mr-Pharma es una aplicación móvil que diseñé y desarrollé utilizando Flutter con el lenguaje de programación Dart. La aplicación tenía como objetivo proporcionar a las farmacias una herramienta práctica para gestionar su inventario de medicamentos farmacéuticos de manera eficiente y organizada.

  • ElectroStock

    ElectroStock es una aplicación móvil que diseñé y desarrollé utilizando Flutter con el lenguaje de programación Dart. La aplicación tuvo como propósito brindar a las empresas de productos electrónicos una herramienta completa para gestionar su inventario de manera eficiente y optimizada.

  • OAutomatizador de Dispensador de Pastillas con C++ y Arduino

    El proyecto "Automatizador de Dispensador de Pastillas" fue una solución que desarrollé utilizando el lenguaje de programación C++ en conjunto con el microcontrolador Arduino y otros elementos electrónicos. El objetivo principal era crear un dispositivo automatizado que facilitara la administración de pastillas para personas con necesidades médicas específicas.
